## HTTPClient

Async HTTP client for the Fluxer REST API. Handles rate limiting, retries on 429/5xx, connection errors, and error mapping.

### Constructor

```python
HTTPClient(
    token: str,
    *,
    is_bot: bool = True,
    api_url: str = "https://api.fluxer.app/v1",
    max_retries: int = 4,
    retry_forever: bool = False
)
```

<ParamField path="token" type="str" required>
  Bot or user token.
</ParamField>

<ParamField path="is_bot" type="bool" default="True">
  Whether this is a bot token (adds "Bot" prefix to auth header).
</ParamField>

<ParamField path="api_url" type="str" default="https://api.fluxer.app/v1">
  Base URL for the API. Use this to connect to self-hosted Fluxer instances.
</ParamField>

<ParamField path="max_retries" type="int" default="4">
  Maximum number of retries for failed requests.
</ParamField>

<ParamField path="retry_forever" type="bool" default="False">
  Whether to retry requests indefinitely.
</ParamField>

### Context Manager Usage

```python
async with HTTPClient(token) as http:
    data = await http.get_channel("123456789")
```

### Core Request Method

#### request

```python
async def request(
    route: Route,
    *,
    json: Any = None,
    data: aiohttp.FormData | None = None,
    params: dict[str, Any] | None = None,
    reason: str | None = None,
    max_retries: int | None = None,
    retry_forever: bool | None = None
) -> Any
```

Make an authenticated request to the Fluxer API.

<ParamField path="route" type="Route" required>
  Route object containing method, path, and parameters.
</ParamField>

<ParamField path="json" type="Any" default="None">
  JSON payload to send.
</ParamField>

<ParamField path="data" type="aiohttp.FormData | None" default="None">
  Form data to send (for file uploads).
</ParamField>

<ParamField path="params" type="dict[str, Any] | None" default="None">
  Query parameters.
</ParamField>

<ParamField path="reason" type="str | None" default="None">
  Audit log reason (sent as X-Audit-Log-Reason header).
</ParamField>

<ParamField path="max_retries" type="int | None" default="None">
  Override the default max_retries for this request.
</ParamField>

<ParamField path="retry_forever" type="bool | None" default="None">
  Override the default retry_forever for this request.
</ParamField>

**Returns:** Parsed JSON response, or None for 204 No Content.

## Rate Limiting

The HTTPClient automatically handles rate limits using Fluxer's response headers:

- `X-RateLimit-Limit` - Max requests in window
- `X-RateLimit-Remaining` - Requests left
- `X-RateLimit-Reset` - Unix timestamp when limit resets
- `X-RateLimit-Reset-After` - Seconds until reset
- `X-RateLimit-Bucket` - Opaque bucket identifier

The client will automatically wait when rate limited and supports both per-route and global rate limits.
