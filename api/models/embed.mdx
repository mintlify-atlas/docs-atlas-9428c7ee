## Overview

The `Embed` class is a builder for creating rich embed objects in Fluxer messages. Embeds allow you to display formatted content with titles, descriptions, fields, images, and more.

## Class Definition

```python
from fluxer.models import Embed

@dataclass
class Embed:
    """Builder for rich embed objects."""
```

## Usage Examples

### Basic Embed

```python
from fluxer import Embed

# Create a simple embed
embed = Embed(
    title="Hello, World!",
    description="This is a basic embed",
    color=0x5865F2  # Blurple color
)

await channel.send(embed=embed)
```

### Embed with Fields

```python
embed = Embed(
    title="Server Statistics",
    description="Current server stats",
    color=0x00FF00
)

embed.add_field(name="Members", value="1,234", inline=True)
embed.add_field(name="Online", value="567", inline=True)
embed.add_field(name="Channels", value="42", inline=True)

await channel.send(embed=embed)
```

### Complete Embed

```python
from datetime import datetime

embed = Embed(
    title="User Information",
    description="Detailed user profile",
    url="https://example.com/profile",
    color=0x5865F2,
    timestamp=datetime.utcnow().isoformat()
)

embed.set_author(
    name="Profile Bot",
    icon_url="https://example.com/bot-icon.png"
)

embed.set_thumbnail(url="https://example.com/avatar.png")

embed.add_field(name="Username", value="CoolUser123", inline=True)
embed.add_field(name="ID", value="123456789", inline=True)
embed.add_field(name="Joined", value="2024-01-01", inline=True)

embed.add_field(
    name="About",
    value="This is a bio that spans the full width",
    inline=False
)

embed.set_image(url="https://example.com/banner.png")

embed.set_footer(
    text="Requested by User",
    icon_url="https://example.com/user-icon.png"
)

await channel.send(embed=embed)
```

### Multiple Embeds

```python
embed1 = Embed(title="Embed 1", description="First embed", color=0xFF0000)
embed2 = Embed(title="Embed 2", description="Second embed", color=0x00FF00)

# Send multiple embeds in one message
await channel.send(embeds=[embed1, embed2])
```

### Method Chaining

```python
embed = (Embed(title="Chained Embed", color=0x5865F2)
    .set_author(name="Bot Name")
    .add_field(name="Field 1", value="Value 1")
    .add_field(name="Field 2", value="Value 2")
    .set_footer(text="Footer text")
)

await channel.send(embed=embed)
```

### Interactive Bot Example

```python
import fluxer
from fluxer import Embed
from datetime import datetime

client = fluxer.Client(token="your_token")

@client.event
async def on_message(message):
    if message.content == "!help":
        embed = Embed(
            title="Bot Commands",
            description="Here are all available commands",
            color=0x5865F2,
            timestamp=datetime.utcnow().isoformat()
        )
        
        embed.set_author(
            name="Help System",
            icon_url=client.user.avatar_url
        )
        
        embed.add_field(
            name="!help",
            value="Display this help message",
            inline=False
        )
        embed.add_field(
            name="!ping",
            value="Check bot latency",
            inline=False
        )
        embed.add_field(
            name="!info",
            value="Get bot information",
            inline=False
        )
        
        embed.set_footer(text=f"Requested by {message.author.display_name}")
        
        await message.reply(embed=embed)
    
    elif message.content == "!serverinfo":
        if not message.guild_id:
            return
        
        guild = await client.fetch_guild(message.guild_id)
        
        embed = Embed(
            title=guild.name,
            description=f"Server ID: {guild.id}",
            color=0x00FF00
        )
        
        if guild.icon_url:
            embed.set_thumbnail(url=guild.icon_url)
        
        embed.add_field(
            name="Owner",
            value=f"<@{guild.owner_id}>",
            inline=True
        )
        embed.add_field(
            name="Members",
            value=str(guild.member_count or "Unknown"),
            inline=True
        )
        embed.add_field(
            name="Created",
            value=guild.created_at.strftime("%Y-%m-%d"),
            inline=True
        )
        
        await message.reply(embed=embed)

client.run()
```

## Limits

- **Title**: Up to 256 characters
- **Description**: Up to 4096 characters
- **Fields**: Up to 25 fields
- **Field Name**: Up to 256 characters
- **Field Value**: Up to 1024 characters
- **Footer Text**: Up to 2048 characters
- **Author Name**: Up to 256 characters
- **Total Characters**: Up to 6000 characters across all fields
