## Overview

The `Channel` class represents any type of Fluxer channel, including text channels, DM channels, voice channels, and categories. It provides methods for sending messages, managing channel data, and connecting to voice channels.

## Class Definition

```python
from fluxer.models import Channel

@dataclass(slots=True)
class Channel:
    """Represents a Fluxer channel (text, DM, voice, category, etc.)."""
```

### send

```python
async def send(
    content: str | None = None,
    *,
    embed: Embed | None = None,
    embeds: list[Embed] | None = None,
    file: File | None = None,
    files: list[File] | None = None,
    message_reference: dict[str, Any] | None = None,
) -> Message
```

Send a message to this channel.

<ParamField path="content" type="str | None">
  Text content of the message
</ParamField>

<ParamField path="embed" type="Embed | None">
  A single embed to include
</ParamField>

<ParamField path="embeds" type="list[Embed] | None">
  Multiple embeds to include
</ParamField>

<ParamField path="file" type="File | None">

<ParamField path="files" type="list[File] | None">
  Multiple File objects to attach
</ParamField>

<ParamField path="message_reference" type="dict[str, Any] | None">

</ParamField>

**Returns:** The created `Message` object

**Examples:**
```python
# Send a simple message
await channel.send("Hello, world!")

# Send with an embed
from fluxer import Embed
embed = Embed(title="Hello", description="World", color=0x5865F2)
await channel.send(embed=embed)

# Send with a file
from fluxer import File
await channel.send("Here's an image", file=File("image.png"))

# Send multiple files
await channel.send(
    "Multiple files",
    files=[File("a.txt"), File("b.txt")]
)

# Send file with embed
await channel.send(
    embed=Embed(title="Data"),
    file=File("data.json")
)
```

## Usage Example

```python
import fluxer
from fluxer import Embed, File

client = fluxer.Client(token="your_token")

@client.event
async def on_message(message):
    if message.content == "!info":
        channel = message.channel
        
        # Create an embed with channel info
        embed = Embed(
            title=f"Channel: {channel.name}",
            color=0x5865F2
        )
        embed.add_field(name="ID", value=str(channel.id))
        embed.add_field(name="Type", value=str(channel.type))
        embed.add_field(name="Created", value=str(channel.created_at))
        
        if channel.topic:
            embed.add_field(name="Topic", value=channel.topic, inline=False)
        
        await channel.send(embed=embed)
    
    elif message.content == "!history":
        # Fetch recent messages
        messages = await message.channel.fetch_messages(limit=10)
        content = "\n".join([f"{m.author}: {m.content[:50]}" for m in messages])
        await message.channel.send(f"Recent messages:\n{content}")

client.run()
```
