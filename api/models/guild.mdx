## Overview

The `Guild` class represents a Fluxer guild (server/community). It provides methods for managing guild resources such as members, roles, emojis, and moderation actions.

## Class Definition

```python
from fluxer.models import Guild

@dataclass(slots=True)
class Guild:
    """Represents a Fluxer guild (server/community)."""
```

## Usage Example

```python
import fluxer

client = fluxer.Client(token="your_token")

@client.event
async def on_ready():
    print(f"Logged in as {client.user}")
    
    # Fetch a guild
    guild = await client.fetch_guild(123456789)
    print(f"Guild: {guild.name}")
    print(f"Created: {guild.created_at}")
    print(f"Members: {guild.member_count}")
    
    # Fetch and display roles
    roles = await guild.fetch_roles()
    for role in roles:
        print(f"Role: {role.name} (Position: {role.position})")
    
    # Create a new role
    mod_role = await guild.create_role(
        name="Moderator",
        color=0x5865F2,
        hoist=True
    )
    
    # Fetch members
    members = await guild.fetch_members(limit=10)
    for member in members:
        print(f"Member: {member.display_name}")

client.run()
```
