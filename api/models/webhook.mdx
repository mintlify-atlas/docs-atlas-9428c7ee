## Overview

The `Webhook` class represents a Fluxer webhook, which allows sending messages to channels without a bot user. Webhooks are useful for integrations, notifications, and automated messages.

## Class Definition

```python
from fluxer.models import Webhook

@dataclass(slots=True)
class Webhook:
    """Represents a Fluxer webhook."""
```

```python
async def send(

    avatar_url: str | None = None,
    file: File | None = None,
    files: list[File] | None = None,

) -> Message | None
```

Send a message with this webhook.

<ParamField path="content" type="str | None">
  Text content of the message
</ParamField>

<ParamField path="embeds" type="list[dict[str, Any]] | None">
  List of embed dicts to include
</ParamField>

<ParamField path="username" type="str | None">
  Override the webhook's default name
</ParamField>

<ParamField path="avatar_url" type="str | None">
  Override the webhook's default avatar
</ParamField>

<ParamField path="file" type="File | None">
  A single File object to attach
</ParamField>

<ParamField path="files" type="list[File] | None">
  Multiple File objects to attach
</ParamField>

<ParamField path="wait" type="bool" default="False">
  If True, returns the created Message
</ParamField>

**Returns:** The created `Message` if `wait=True`, otherwise None

**Examples:**
```python
# Simple message
await webhook.send("Hello from webhook!")

# Custom username and avatar
await webhook.send(
    "Custom message",
    username="Custom Bot",
    avatar_url="https://example.com/avatar.png"
)

# With embed
from fluxer import Embed
embed = Embed(title="Notification", description="Something happened!")
await webhook.send(embeds=[embed.to_dict()])

# With file
from fluxer import File
await webhook.send(
    "Here's a file",
    file=File("document.pdf")
)

# Wait for message response
message = await webhook.send("Test", wait=True)
print(f"Message ID: {message.id}")
```

### Creating and Using a Webhook

```python
import fluxer

client = fluxer.Client(token="your_token")

@client.event
async def on_ready():
    # Create a webhook in a channel
    channel = await client.fetch_channel(123456789)
    
    # Note: You'd use the HTTP client to create a webhook
    # webhook = await channel.create_webhook(name="My Webhook")
    
    # Send a message via webhook
    await webhook.send(
        "Hello from webhook!",
        username="Notification Bot",
        avatar_url="https://example.com/icon.png"
    )

client.run()
```

### GitHub-Style Notifications

```python
from fluxer import Embed
import fluxer

async def send_github_notification(webhook, repo, action, user):
    """Send a GitHub-style webhook notification."""
    
    embed = Embed(
        title=f"[{repo}] {action}",
        description=f"Action performed by {user}",
        color=0x238636  # GitHub green
    )
    
    embed.set_author(
        name=user,
        icon_url="https://github.com/user.png"
    )
    
    embed.add_field(
        name="Repository",
        value=repo,
        inline=True
    )
    embed.add_field(
        name="Action",
        value=action,
        inline=True
    )
    
    await webhook.send(
        embeds=[embed.to_dict()],
        username="GitHub",
        avatar_url="https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png"
    )

# Usage
await send_github_notification(
    webhook,
    repo="user/repo",
    action="Push to main",
    user="developer123"
)
```

### Monitoring System

```python
from fluxer import Embed
from datetime import datetime

async def send_alert(webhook, level, message, details=None):
    """Send a monitoring alert via webhook."""
    
    colors = {
        "info": 0x5865F2,    # Blue
        "warning": 0xFEE75C, # Yellow
        "error": 0xED4245,   # Red
        "success": 0x57F287  # Green
    }
    
    embed = Embed(
        title=f"{level.upper()}: {message}",
        description=details or "No additional details",
        color=colors.get(level, 0x99AAB5),
        timestamp=datetime.utcnow().isoformat()
    )
    
    embed.set_footer(text="Monitoring System")
    
    await webhook.send(
        embeds=[embed.to_dict()],
        username="Monitor Bot"
    )

# Usage
await send_alert(
    webhook,
    level="error",
    message="High CPU Usage",
    details="CPU usage exceeded 90% for 5 minutes"
)
```

## Security Notes

- **Keep webhook tokens secret**: Anyone with the webhook URL and token can send messages
- **Use HTTPS**: Always use secure connections when sending webhook data
- **Validate inputs**: If accepting user input for webhook messages, sanitize and validate it
- **Rate limits**: Webhooks are subject to rate limits (check Fluxer API documentation)
